# Quick Commands

Replace `~/pdf-reading-tracker` with wherever you cloned the repo.

## Go to project
```bash
cd ~/pdf-reading-tracker
```

## Run the processor (input folder)
```bash
python3 -m app.main
```

## Run with specific files
```bash
python3 -m app.main path/to/file.pdf path/to/notes.txt
```

## Open master reading list
```bash
open output/reads.csv
```

## Open category index
```bash
open output/index.csv
```

## Open category folder
```bash
open output/by-category
```

## Open a specific category file
```bash
open output/by-category/ai.csv
open output/by-category/philosophy.csv
open output/by-category/university.csv
open output/by-category/society.csv
open output/by-category/software.csv
```

## Open project in VS Code
```bash
code .
```
